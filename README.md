## 🌸 About

Petal is a modern Pomodoro timer built to make productivity feel less intimidating and more enjoyable.

Instead of being just another countdown clock, Petal weaves together focus sessions, task management, daily goals, productivity insights, and achievements — all wrapped in a calming, soft-pink aesthetic.

Whether you're studying, coding, reading, or working, Petal helps you stay consistent while keeping the experience beautiful and distraction-free.

## 🌱 Getting Started

```bash
# Clone the repository
git clone https://github.com/yourusername/petal.git

# Move into the project
cd petal

# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```

## 🌼 Why Petal?

Most Pomodoro timers only count time.

Petal focuses on creating an *experience* — one that encourages consistency through thoughtful design, habit tracking, and gentle visual feedback. Because staying productive shouldn't feel overwhelming.
