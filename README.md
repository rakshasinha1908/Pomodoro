# 🍅 Pomodoro Timer — Focus & Flow

A minimal, aesthetic Pomodoro Timer built for myself to **track study & work sessions effectively** and stay consistent without burnout.

This project is not just a timer — it’s a productivity companion designed to keep me in flow while giving visual feedback, session tracking, and smooth interactions.

---

## ✨ Features

* ⏱ **Three Modes**

  * Focus
  * Short Break
  * Long Break (auto after every 4 focus sessions)

* 🔁 **Smart Auto Switching**

  * Focus → Short Break
  * Every 4 Focus → Long Break

* 📊 **Session Tracking**

  * Counts completed Focus, Short, and Long sessions
  * “Sessions Today” counter
  * Reset all sessions anytime

* ⏲ **Custom Time Controls**

  * Increase / decrease time for each mode
  * Changes apply instantly when paused

* 💾 **Persistent Data**

  * Saves mode, time, and session data in `localStorage`
  * Resume exactly where you left off
